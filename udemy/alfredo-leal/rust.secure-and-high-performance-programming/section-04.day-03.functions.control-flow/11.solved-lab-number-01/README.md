## Explicación del Código

### Declaración de Variables:

- **String:** Usamos `String::from` para inicializar una cadena dinámica.
- **f32 y u32:** Definimos el precio y la cantidad como tipos apropiados para sus valores.
- **bool:** Un tipo escalar para representar verdadero o falso.

---

### Cálculos con Variables:

- Multiplicamos el precio por la cantidad para obtener el valor total del inventario.
- Si el producto está en oferta (`on_sale`), ajustamos el precio y recalculamos el valor total.

---

### Conversiones y Formateo:

- Convertimos el tipo `f32` a `f64` usando `as`.
- Formateamos la cantidad disponible como una cadena con `format!`.

---

### Impresión Formateada:

- Usamos `println!` con *placeholders* (`{}` y `{:.2}`) para mostrar los valores con formato.
