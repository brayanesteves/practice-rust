Rust es un lenguaje de programación relativamente nuevo, fue lanzado en el 2015 y e sun lengauje de programación de la línea C y C++. Algunos sistemas operativos se estan construyendo en este lenguaje de programación Rust y se dice que puede reemplazar a C y C++, ya que Rust hace un enfási muy grande, en la seguridad y el uso de la memoria.
Rust es uno de los lenguajes de programación más amado por la comunidad, según una encuesta hecha en StackOverflow y cada año está tomando más popularidad este lenguaje de programación.
Es fundamental que sepas programar en Rust, puedes que en el futuro lo puedas necesitar, si vas a enfrentar un proyecto en Rust y neceistas aprender este lenguaje de programación.
