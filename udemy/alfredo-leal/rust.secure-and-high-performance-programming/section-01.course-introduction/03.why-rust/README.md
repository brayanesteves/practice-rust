¿Porque es importante aprender este lengauje de programación Rust?
Rust es un lenguaje de programación que se usa para crear software confiable y eficiente.
Es una buena opción para programar:
1) Sistemas.
2) Videojuegos.
3) Dispositivos integrados.
4) Aplicaciones Web.

Algunas de las razones principales para desarrollar en Rust es:
1) Seguridad en Memoria: Rust garantiza la seguridad de la memoria, sin sacrificar el rendimiento, Rust de la da mucho enfásis a este concepto de la memoria.
2) Eficiencia: Es uno de los motivos, Rust es rápido y eficiente con la memoria.
3) Concurrencia: Rust permite la programación multihilo, sin riesgo de acceder a memoria corrupta.
4) Herramientas: Las herramientas que provee Rust estan adaptadas y estan preocupadas del uso de la memoria.
5) Software Libre: Otra razón para programar en Rust, es de Software libre, es de código abierto, se puede usar libremente sin pagar licencia.
6) Sintaxis: La sintaxis es similar a la de C/C++.
7) Provee opciones de tamaño especifico para enteros y números de puntos flotantes.
8) Comunidad: Es una de las más destacadas en el mundo de software debido a su enfoque en la colaboración abierta, seguridad y apoyo mutuo.

Características imporetantes:
1) Elimina muchos tipos de errores durante la compilación.
2) Permite crear bibliotecas.
3) Tiene un sistema de seguridad de memoria.
4) Minimiza la sobrecarga en tiempo de ejecución.

Dato Curioso
Rust fue lanzado en 2015, pero su desarrollo comenzó en 2006 como proyecto de Graydon Hoare.
Mozilla lo respaldó en 2009 por su enfoque en seguridad de memoria y concurrencia. Tras una fase experimental, se abrió al público en 2010 y su versión estable llegó en 2015.
