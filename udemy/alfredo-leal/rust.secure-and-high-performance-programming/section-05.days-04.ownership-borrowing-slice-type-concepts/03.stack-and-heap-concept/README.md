**Stack y Heap.**

Son dos áreas principales de la memoria que se utilizan para almacenar datos.
Stack.

### **Stack**

El **Stack** es un estructura de datos en la memoria que sigue el principio de último en entrar, primero en salir.

**1)** Inserta un nuevo elemento en la parte superior del **Stack** (**Push**).

**2)** Retirar el elemento que está en la parte superior del stack (**Pull**).

**Importante:**
Los tipos de datos que se guarden deben tener un tamaño conocido.

### **Heap**

El **Heap** es un área de memoria más grande y flexible que se usa para almacenar datos cuyo tamaño no es conocido o es muy grande para el **Stack**.

Se diferencian en que almacenar información en el **Stack** es mucho más rápido.
