Ownership.
Es un sistema que gestiona la memoria y asegura la seguridad en tiempo de compilación.
La memoria se administra a través de un sistema de propiedad.
Reglas que verifica el compilador.
1) Cada valor **Rust** tiene un propietario.
2) Cada valor en **Rust** tiene un único propietario a la vez.
3) El propietario se elimina cuando queda fuera del alcance del valor.
